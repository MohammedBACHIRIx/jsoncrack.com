# JSON Crack

## Core Identity
A visual tool for viewing and editing JSON data as an interactive graph. It must empower developers to parse complex data structures quickly.

## Target Audience
Developers, data engineers, and technical professionals analyzing nested configurations or API responses.

## UI Register: The Professional Workbench
- **Purpose**: Maximize focus on the content graph and JSON editor.
- **Vibe**: Clean, flat surfaces, subtle grays and blues. Highly functional, unobtrusive.
- **Rule**: Content > Chrome. The application chrome must recede into the background. Depth is strictly functional (e.g. draggable nodes, active menus).
